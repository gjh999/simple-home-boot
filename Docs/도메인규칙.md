# 공공데이터 공통표준 — 도메인 규칙

> 출처: 공공데이터 공통표준(2025.11월) | 공통표준도메인 시트
> 총 129개 도메인

## 개요

도메인은 속성이 가질 수 있는 값의 집합을 정의한다. 컬럼명 말미의 **형식단어**와 1:1로 대응하며,
데이터타입·길이·표현형식을 표준화한다.

**도메인명 표기 규칙**: `분류명 + 데이터타입약어 + 길이` (예: `금액N15`, `명V100`, `여부C1`)

| 약어 | 데이터타입 | 설명 |
|------|-----------|------|
| N | NUMERIC | 숫자 |
| V | VARCHAR | 가변길이 문자 |
| C | CHAR | 고정길이 문자 |
| D | DATE | 날짜 |
| T | TIMESTAMP | 일시 |
